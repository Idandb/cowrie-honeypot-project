KB003: Password Reset Policy and Procedure
-----------------------------------------
Policy:
- All CompanyServer user passwords must be reset every 90 days.
- Minimum password length: 12 characters.
- Must include: uppercase, lowercase, numbers, and special characters.
- Cannot reuse any of the last 5 passwords.

Self-Service Password Reset (if enabled):
- Navigate to https://passwordreset.companyserver.com
- Follow on-screen instructions. Requires answering security questions or OTP via registered email/phone.

Manual Password Reset (via IT Support):
- User must contact IT Support via phone or in person (for verification).
- A temporary password will be provided.
- User will be required to change the temporary password upon first login.

Security Reminder: Do not share your password with anyone, including IT Support. IT Support will never ask for your current password.
